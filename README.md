koa-starter
===========

A simple Koajs application starter template. 

quick start
=============

**Clone koa-starter:**

```sh
$ git clone https://github.com/justmyfreak/koa-starter.git
```

**Install dependencies:**

```sh
$ npm install
```

**Run koa-starter:**

```sh
$ npm start
```

or 

```sh
$ node --harmony index.js
```
Open [http://localhost:3000](http://localhost:3000)

**Run the test:**

```sh
$ npm test
```
